TGMF: Taylor Gaussian Motion Field for Dynamic Head Avatars

A Taylor-based local motion field that explicitly models Gaussian motion with interpretable velocity and acceleration, and refines it with bidirectional Gaussian--mesh optimization for stable dynamic head avatars.

Watch Demo Subject Gallery Do / Not Do Citation

Overview

TGMF models local motion around a reference state instead of treating deformation as a frame-conditioned black box. The method uses a second-order Taylor formulation for Gaussian position, scale, and rotation, then couples the resulting motion field with a mesh scaffold through differentiable feedback.

This page is designed for project presentation on GitHub and GitHub Pages, with a compact demo-first layout similar in spirit to existing avatar repositories such as MeGA, but centered on TGMF's motion-modeling contribution.

Do

  • Show smooth local motion around a reference facial state.
  • Emphasize mouth, eyelid, contour, and hair-boundary consistency.
  • Present novel-view synthesis and novel-expression / self-reenactment results.
  • Explain that TGMF is an interpretable motion model with velocity and acceleration terms.
  • Use the demo page to show stable temporal transitions rather than only single frames.

Not Do

  • Do not describe TGMF as a universal solution for arbitrarily long or highly oscillatory motion.
  • Do not oversell very high-frequency micro-motion fitting.
  • Do not present the Taylor field as a global video-time lookup table.
  • Do not imply that all nine subject videos are already online.
  • Do not bury the video behind the code section; keep the demo visible near the top.
